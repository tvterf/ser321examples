You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=10&num2=10 -- returns the product of num1 and num2
  • /github?query=users/username/repos -- returns the full_name, id, and login of the repos
  • /hash?word1=tyler&word2=terhufen -- returns the hashcode of the 2 words...yes very creative and useful I know ;)
  • /futureValue?pVal=10000&iRate=8&periods=5 -- returns the future value of the pVal with the given interest rate and period
